# 📘 ICT WITH MALINDA JAYASIRI

A professional and modern website for **ICT O/L Students**, designed and developed using **HTML, CSS, JavaScript, and Bootstrap**.  
This project is created for **Malinda Jayasiri**, who teaches ICT for G.C.E. O/L students, to provide class information, exam preparation tips, and contact details online.

---

## 🚀 Features

- 🎯 **Hero Section** with countdown timer to O/L exams  
- 📖 **About Section** highlighting ICT learning and benefits  
- 📅 **Exam Countdown Timer** for motivation and tracking  
- 📩 **Contact Section** with email form and details  
- 🔗 Social media integration (Facebook, Instagram, LinkedIn, etc.)  
- 📱 Fully **responsive design** (mobile, tablet, desktop)  

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript  
- **Frameworks/Libraries:** Bootstrap, AOS (Animate on Scroll)  
- **Icons:** Bootstrap Icons  
- **Version Control:** Git & GitHub
